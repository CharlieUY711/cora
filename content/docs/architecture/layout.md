# Ancho del ecosistema CORE

**Valor:** 1152px (Tailwind: max-w-6xl)
**Variable CSS:** --max-w

## Aplica a
Absolutamente todas las apps del ecosistema — actuales y futuras:
- core-hub, core-biblio, core-foundation
- core-landing, core-marketing
- core-market (incluido modal de login)
- Cualquier app futura

## Uso en CSS
max-width: var(--max-w);
margin: 0 auto;

## Uso en Tailwind
className="max-w-6xl mx-auto w-full px-8"

IMPORTANTE: No cambiar este valor sin actualizar todas las apps.
